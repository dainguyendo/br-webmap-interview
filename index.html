ArcGIS JS API Web Map App

This web map application was created for a technical interview at Blue Raster. The assignment was done utilizing Esri's JavaScript 4.0 API. The goal of the assignment was to demonstrate a familiarity with the API and demonstrate various capabilities of myself in the API environment. At the top of the page, quick links are available to the github repository and the JS API documentation.

    What's included:
  • An interactive map with an initial view of depicting proportions of Hispanic Population relative to the Total.
  • A "Home" button that recenter's the view to Washington, D.C.
  • A toggle basemap widget.
  • Zoom in close enough to the Blue Raster Office or click on the logo and a graphic symbol will appear.
  • When clicking a point on the map a popup window displaying lat/long appears.
